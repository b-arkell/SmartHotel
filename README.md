# SmartHotel
Group 6 project 3

## Setup Instructions
1. Run the backend sln in visual studio
2. Open the frontend in VS code
3. Navigate to the frontend folder
4. Run the command "npm install"
5. If there is an error, open powershell as an Admin and run "Set-ExecutionPolicy RemoteSigned"
6. Run the backend using F5
7. Run the frontend using "npm start"
8. Download any additional dependecies requested

Note: If there is an error with the react app page, add a .env file to your main smartHotel folder. In that file put "REACT_APP_API_URL=http://localhost:7194/" with the local host # replaced with the one given to you from the backend terminal page.
